NatashaClub Kiss Sender
===========

###### Current version 1.6-beta1

##### Используемые технологии, фреймворки и библиотеки
* Java8
* CDI - Dagger2
* HTTP client - OkHttp3 + Retrofit
* Html parser - JSoup
* DB - Postgres, H2DB(embedded), JDBI
* GUI - JFoenix
